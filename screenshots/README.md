# Screenshots

Add your screenshots here. Naming convention:

- `dashboard.png`   → Main dashboard with KPI cards
- `fleet.png`       → Vehicle registry / fleet management page
- `drivers.png`     → Driver management with license status
- `trips.png`       → Trip dispatch and management
- `maintenance.png` → Maintenance logs
- `fuel.png`        → Fuel & expense logs
- `analytics.png`   → Analytics charts and reports
- `login.png`       → Login page with demo accounts

## How to take screenshots

1. Run both servers (see main README)
2. Open http://localhost:3001
3. Login with any demo account
4. Use Snipping Tool (Win+Shift+S) or browser DevTools > Device Toolbar
5. Recommended size: 1440×900 or 1280×800
